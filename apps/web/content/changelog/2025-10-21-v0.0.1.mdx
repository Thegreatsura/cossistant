## Widget

- **Initial Release**: First public version of the Cossistant React SDK
- **Headless Architecture**: Flexible hooks and primitives for building custom support widgets
- **Real-time Communication**: WebSocket-driven messaging with presence tracking
- **Pre-built Components**: Complete support widget with Tailwind styling
- **TypeScript-First**: Full type safety throughout the SDK
- **Conversation Management**: Hooks for managing conversations, messages, and timeline events
- **File Uploads**: Drag-and-drop file upload support with presigned URLs
- **Markdown Support**: Rich message formatting with react-markdown
- **Accessibility**: WCAG-compliant components with proper ARIA labels
- **Next.js Support**: Dedicated `@cossistant/next` package with SSR optimization
